# RentRadar API App

中文：这是 RentRadar 的 API 应用，基于 `Next.js Route Handlers`。  
English: This is the RentRadar API app, implemented with `Next.js Route Handlers`.

## This package contains

- Health check
- Upload signing and storage routes
- Manual image analysis
- Live scan analysis
- Intelligence and negotiation routes
- Knowledge base query
- Listing discovery / extraction
- Reverse geocoding
- Static map generation
- MiniMax TTS alert route
- 3D room reconstruction route

## Run only the API app

```bash
pnpm --filter api dev
```

Default local URL:

- `http://localhost:3001`

## Build only the API app

```bash
pnpm --filter api build
```
